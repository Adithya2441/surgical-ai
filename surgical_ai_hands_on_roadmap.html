4-Stage Project Roadmap — From Baseline to SAIS-Like System
Each stage builds on the last. By stage 4 you'll have built something structurally close to SAIS on a public benchmark — and you'll have genuine things to talk about in interviews.
1
Frame-Level CNN Baseline
Week 1 · JIGSAWS · ~10 hours of work
Start here
What you build: A ResNet50 that classifies individual frames from JIGSAWS into one of 10 gesture classes. No temporal modeling — just frame-by-frame CNN classification.
Why start here: This is the naive baseline. When you add temporal modeling in Stage 2 and see performance jump, you'll viscerally understand why SAIS uses a Transformer.
What you'll learn: Video data pipelines, frame extraction, surgical dataset quirks, basic evaluation with F1 score, leave-one-user-out cross validation (the JIGSAWS standard protocol)
Deliverables
✓ JIGSAWS dataloader in PyTorch
✓ Frame extraction from .avi files
✓ ResNet50 transfer learning classifier
✓ Leave-one-user-out evaluation
✓ F1 score per gesture class
✓ Confusion matrix visualization
Expected F1: ~70–75% (SAIS got 87.5% — the gap tells you what temporal modeling adds)
2
Add Optical Flow + Temporal Modeling
Week 2–3 · JIGSAWS · ~15 hours
What you build: Two upgrades — (1) compute optical flow between frames using RAFT (exact method from SAIS paper), and (2) replace single-frame classification with a sliding window of T=8 frames fed through an LSTM or simple Transformer.
The aha moment: Watch performance jump when you add temporal context. Then try removing optical flow and watch it drop slightly. You'll have personally reproduced the key ablation from the paper.
Deliverables
✓ RAFT optical flow extraction pipeline
✓ Clip-level dataloader (T frames per sample)
✓ LSTM baseline for temporal modeling
✓ Simple Transformer encoder (2–4 layers)
✓ Ablation: RGB only vs flow only vs both
Expected F1: ~82–86% (getting close to SAIS)
3
DINO-ViT Features + Contrastive Learning
Week 3–4 · JIGSAWS · ~15 hours
What you build: Replace ResNet50 with a frozen DINO-pretrained ViT-Small from timm. Add learnable prototypes per gesture class. Train with InfoNCE loss instead of cross-entropy. This is structurally SAIS on JIGSAWS.
Bonus: Visualize the attention maps from the ViT using DINO's built-in attention visualization — you'll see it attend to instrument tips, just like Figure 6 of the paper.
Deliverables
✓ DINO-ViT feature extractor (frozen)
✓ Learnable prototype vectors per class
✓ InfoNCE / SupCon loss implementation
✓ Attention map visualization
✓ t-SNE plot of learned embedding space
Expected F1: ~87–90% (at or above SAIS level)
4
Fine-tune a Surgical Foundation Model (ZEN-style)
Week 4–6 · JIGSAWS + Cholec80 · ~20 hours
Advanced
What you build: Take a public surgical foundation model checkpoint (SurgicalSAM, SurgFormer, or UniSurg) and fine-tune it on JIGSAWS in few-shot fashion — using only 5–10 examples per class. Compare against your Stage 3 model trained with full data.
The point: You'll personally experience why foundation models matter. The few-shot foundation model will match or beat your fully-trained Stage 3 model — with 100x less labeled data.
Deliverables
✓ Load pretrained surgical foundation model
✓ Few-shot fine-tuning loop
✓ Label efficiency curve (1, 5, 10, 50, all)
✓ Zero-shot probe of representation quality
✓ Side-by-side comparison vs Stage 3
Interview gold: "I reproduced the few-shot advantage of foundation models on JIGSAWS"
Code Starter 1 — JIGSAWS Dataloader
The hardest part of any project is the data pipeline. This gets you loading JIGSAWS clips in PyTorch.
import torch from torch.utils.data import Dataset import cv2, os, numpy as np class JIGSAWSDataset(Dataset): def __init__(self, video_dir, label_dir, clip_len=8, transform=None): self.clips = [] self.labels = [] self.clip_len = clip_len self.transform = transform # Gesture vocabulary from the paper self.gesture_map = { 'G1':0, 'G2':1, 'G3':2, 'G4':3, 'G5':4, 'G6':5, 'G8':6, 'G9':7, 'G10':8, 'G11':9 } for label_file in os.listdir(label_dir): if not label_file.endswith('.txt'): continue video_id = label_file.replace('_capture1.txt', '') video_path = os.path.join(video_dir, video_id + '_capture1.avi') with open(os.path.join(label_dir, label_file)) as f: for line in f: start, end, gesture = line.strip().split() if gesture in self.gesture_map: self.clips.append((video_path, int(start), int(end))) self.labels.append(self.gesture_map[gesture]) def __len__(self): return len(self.clips) def __getitem__(self, idx): video_path, start, end = self.clips[idx] label = self.labels[idx] cap = cv2.VideoCapture(video_path) frames = [] # Sample clip_len frames uniformly from the gesture segment frame_ids = np.linspace(start, end, self.clip_len, dtype=int) for fid in frame_ids: cap.set(cv2.CAP_PROP_POS_FRAMES, fid) ret, frame = cap.read() if ret: frame = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB) frame = cv2.resize(frame, (224, 224)) frames.append(frame) cap.release() frames = np.stack(frames).astype(np.float32) / 255.0 frames = torch.tensor(frames).permute(0, 3, 1, 2) # T,C,H,W return frames, label
Code Starter 2 — DINO-ViT Feature Extractor (MPS)
Load the exact same ViT used as SAIS's frozen backbone, run on MPS, extract frame features.
import torch, timm from torchvision import transforms device = torch.device("mps" if torch.backends.mps.is_available() else "cpu") # Load DINO-pretrained ViT-Small — the exact backbone used in SAIS vit = timm.create_model( 'vit_small_patch16_224_dino', pretrained=True, num_classes=0 # remove classification head → get 384-dim features ).to(device) vit.eval() # Freeze all weights — SAIS keeps ViT frozen for param in vit.parameters(): param.requires_grad = False # ImageNet normalization (DINO was trained on ImageNet) normalize = transforms.Normalize( mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225] ) def extract_frame_features(frames): """ frames: Tensor of shape (T, C, H, W) — one clip returns: Tensor of shape (T, 384) — one feature per frame """ with torch.no_grad(): frames = normalize(frames.to(device)) # Process all T frames as a batch features = vit(frames) # (T, 384) return features # Visualize DINO attention maps (shows what ViT focuses on) def get_attention_map(frame_tensor): """Returns spatial attention from last ViT layer""" attentions = vit.get_intermediate_layers( frame_tensor.unsqueeze(0).to(device), n=1 ) return attentions # visualize to see instrument focus
Code Starter 3 — Temporal Transformer Encoder
The temporal module from SAIS — takes T frame features, outputs a single clip-level representation via CLS token.
import torch.nn as nn class TemporalTransformer(nn.Module): def __init__(self, feat_dim=384, n_heads=8, n_layers=4, dropout=0.1): super().__init__() self.feat_dim = feat_dim # Learnable CLS token — aggregates clip-level info self.cls_token = nn.Parameter(torch.randn(1, 1, feat_dim)) # Learnable positional embeddings (one per frame position) self.pos_embed = nn.Parameter(torch.randn(1, 65, feat_dim)) # 64 + CLS # Stack of Transformer encoder layers encoder_layer = nn.TransformerEncoderLayer( d_model=feat_dim, nhead=n_heads, dim_feedforward=feat_dim * 4, dropout=dropout, batch_first=True ) self.transformer = nn.TransformerEncoder(encoder_layer, n_layers) self.norm = nn.LayerNorm(feat_dim) def forward(self, x): """ x: (B, T, feat_dim) — batch of clips, T frame features each returns: (B, feat_dim) — one clip representation per item """ B = x.shape[0] # Prepend CLS token to each clip in the batch cls = self.cls_token.expand(B, -1, -1) # (B, 1, D) x = torch.cat([cls, x], dim=1) # (B, T+1, D) # Add temporal positional embeddings x = x + self.pos_embed[:, :x.shape[1], :] # Self-attention across all frames simultaneously x = self.transformer(x) # every frame attends to every other frame x = self.norm(x) # CLS token at position 0 = clip-level summary return x[:, 0, :] # (B, feat_dim)
Code Starter 4 — RAFT Optical Flow
Extract optical flow the same way SAIS does — using RAFT from torchvision.
from torchvision.models.optical_flow import raft_small, Raft_Small_Weights import torch device = torch.device("mps" if torch.backends.mps.is_available() else "cpu") # Load pretrained RAFT — same model referenced in SAIS paper weights = Raft_Small_Weights.DEFAULT raft = raft_small(weights=weights).to(device).eval() def compute_flow(frame1, frame2): """ frame1, frame2: Tensors (1, C, H, W) — consecutive frames returns: flow map (1, 2, H, W) — dx and dy per pixel """ with torch.no_grad(): # RAFT expects uint8 images [0, 255] as input f1 = (frame1 * 255).byte().to(device) f2 = (frame2 * 255).byte().to(device) flow_predictions = raft(f1, f2) return flow_predictions[-1] # final refined flow estimate def extract_clip_flows(frames): """ frames: Tensor (T, C, H, W) returns: flow_maps (T-1, 2, H, W) """ flows = [] for i in range(len(frames) - 1): flow = compute_flow(frames[i:i+1], frames[i+1:i+2]) flows.append(flow.squeeze(0)) return torch.stack(flows)
Code Starter 5 — InfoNCE Loss with Prototypes
The exact classification mechanism SAIS uses — learnable prototypes + cosine similarity + InfoNCE.
import torch, torch.nn as nn, torch.nn.functional as F class PrototypeClassifier(nn.Module): def __init__(self, feat_dim=384, n_classes=10, temperature=0.07): super().__init__() self.temperature = temperature # One learnable prototype per class — trained end-to-end self.prototypes = nn.Parameter(torch.randn(n_classes, feat_dim)) def forward(self, h, labels=None): """ h: (B, feat_dim) — clip embeddings from Temporal Transformer labels: (B,) — gesture class indices (None at inference) """ # L2-normalize embeddings and prototypes h_norm = F.normalize(h, dim=-1) p_norm = F.normalize(self.prototypes, dim=-1) # Cosine similarity to all prototypes: (B, n_classes) logits = h_norm @ p_norm.T / self.temperature if labels is not None: # InfoNCE = cross-entropy in this formulation loss = F.cross_entropy(logits, labels) return loss, logits return logits # Full SAIS-like model assembly class SAISLite(nn.Module): def __init__(self, n_classes=10): super().__init__() self.temporal = TemporalTransformer(feat_dim=384) self.proj = nn.Sequential( nn.Linear(384, 256), nn.ReLU(), nn.Linear(256, 256) ) self.classifier = PrototypeClassifier(feat_dim=256, n_classes=n_classes) def forward(self, frame_features, labels=None): # frame_features: (B, T, 384) from frozen DINO-ViT h = self.temporal(frame_features) # (B, 384) h = self.proj(h) # (B, 256) return self.classifier(h, labels)
